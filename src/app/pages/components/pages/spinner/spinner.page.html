Circular indicator of progress or an ongoing task. Also known as: "loader", "progress spinner". Not to be confused with Progress Circle.

Styling Spinners Colors

Spinner color determines which color scheme the spinner should use. There are 8 spinner colors available:

Color Description
none The uncolorized (white or black) version of the spinner.
prmiary The primary color of the spinner (default).
secondary The secondary color of the spinner.
danger This color signifies dangerous or destructive actions.
warn This color signifies a warning.
success This color signifies positive actions.
info This color should be used when paired with an informational message.
currentColor The spinner will inherit its color from the current text color.
Sizing the spinner

Spinners can be sized using CSS, by setting its width property. The height will adjust automatically, keeping it square. By default, the spinner's width is 2rem.

Providing default values

Default values of all spinner properties can be provided using the provideSpinnerDefaults provider function. It takes an object of new default values, and returns a provider token.
