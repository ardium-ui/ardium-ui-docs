Divider separates related content into clearer sections. It supports horizontal and vertical orientations, optional inline content, and directive usage on existing elements.

Variants

Use the default full variant for full-width lines, or middle to add inset spacing.

Directive usage

Apply ard-divider to existing elements when you need divider behavior without introducing another component element.

Vertical flow in flex layouts

Set vertical and flexItem to make the divider stretch naturally inside a flex row.

Content alignment

Divider content can be aligned with textAlign using start, center, or end.

Directive with content

The directive variant can also render content between lines, useful for list sections and grouped items.

Styling customization

Customize divider visuals through CSS variables such as width, style, color, and alignment inset values.

Providing default values

Use provideDividerDefaults to configure global defaults for divider behavior in your app.
