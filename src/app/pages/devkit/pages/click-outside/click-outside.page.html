Directive for detecting when the user clicks outside a given element.

Listening to the event

ClickOutisdeDirective provides an event emitter (ardClickOutside) that emits a click event or a touch event every time the user clicks outside of a given element.

Internally the directive listens to mousedown and touchstart events on the whole document element. In that case stopping event propagation might not prove useful in standard circumstances. If the event needs to be stopped from firing, define an event listener with capture option set to true and stop it there.

Performance considerations

The directive internally uses Node.contains(), which, when executed repeatedly on particularly large DOM trees, may lead to performance issues. It's recommended to avoid using the directive if more than 30 clicks per second are expected, as performance may degrade depending on the user's device. For most cases with fewer interactions, no significant issues should arise.
