The Counter Signal is a reactive signal designed to handle numeric values, providing convenient mutation helpers for incrementing, decrementing, and resetting the counter. It also exposes computed properties to query the counter's state.

Counter signal methods

The writable counter signal exposes methods for mutating the counter value, including increment(), decrement(), set(), and update(). These methods enable reactive updates to the counter and automatically trigger Angular's reactivity engine.

The increment() and decrement() methods allow you to modify the counter by a specified amount, with the default value being 1. The set() method allows setting the counter to an explicit value, while update() lets you modify the counter using an updater function.

Resetting and base value

The counter includes a reset() method that resets the counter back to its initial value. The base value can be customized using the setBaseValue() method, which allows defining a new initial value for the counter.

Whenever the counter is reset, the isReset property is updated to true. isReset will change to false after any update to the signal value and will only change back to true after the signal is reset again.

Read-only counter signal

The asReadonly() method provides a read-only version of the counter signal. This version disables all mutation methods and provides access to the read functions isReset() and value(), ensuring that external code can observe the counter value without modifying it.
