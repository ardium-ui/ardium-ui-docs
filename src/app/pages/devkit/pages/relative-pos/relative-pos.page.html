The EventRelativePos class calculates the relative position of a pointer event (mouse or touch) within an HTML element. It determines whether the event overflows beyond the element’s boundaries and provides an easy-to-use interface for accessing this data.

Calculating Relative Position

The class computes the relative position of an event based on its coordinates in relation to an element’s bounding rectangle. The result is represented with the top, right, bottom, and left properties, which indicate how far the event occurred from each respective edge of the element.

Additionally, boolean properties such as overflowsTop or overflowsRight indicate whether the event is outside the element’s boundaries in a specific direction.

The overflow properties provide insight into whether an event is outside the element's boundaries. The overflows property is a combined check that returns true if the event position extends beyond any of the four edges.
