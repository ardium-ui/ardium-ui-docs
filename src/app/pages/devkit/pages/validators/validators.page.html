Additional validators supplement the standard Angular validators with more specific use cases.
Prerequisites

The extended validators require no additional setup, and they can be used in the same way as the standard Angular validators. They are all available as static methods on the ExtValidators class.

Available validators

The extended validators include the following validators:

Text validators
ExtValidators.contains

Validates that the control's value contains a specific substring.

ExtValidators.notContains

Validates that the control's value does not contain a specific substring.

ExtValidators.lowercase

Validates that the control's value is all lowercase.

ExtValidators.uppercase

Validates that the control's value is all uppercase.

ExtValidators.namedPattern

Validator that wraps the built-in Validators.pattern but allows specifying a custom error name instead of the default pattern key.

Number validators
ExtValidators.divisibleBy

Validates that the control's value is divisible by a specific number.

Character set validators
ExtValidators.ascii

Validates that the control's value contains only ASCII characters.

ExtValidators.base64

Validates that the control's value is a valid base64 string.

ExtValidators.hex

Validates that the control's value is a valid hexadecimal string.

ExtValidators.octal

Validates that the control's value is a valid octal string.

ExtValidators.binary

Validates that the control's value is a valid binary string.

IT validators
ExtValidators.ipAddress

Validates that the control's value is a valid IP address (IPv4 or IPv6).

ExtValidators.uuid

Validates that the control's value is a valid UUID.

ExtValidators.hexColor

Validates that the control's value is a valid hexadecimal color string.

ExtValidators.rgbColor

Validates that the control's value is a valid RGB color string.

ExtValidators.hslColor

Validates that the control's value is a valid HSL color string.

ExtValidators.password

Validates that the control's value is a valid password, with various requirement options.

Array validators
ExtValidators.arrayContains

Validates that a FormControl array contains a specific value.

ExtValidators.arrayNotContains

Validates that a FormControl array does not contain a specific value.

ExtValidators.arrayUnique

Validates that all values in a FormControl array are unique.

File validators
ExtValidators.fileMaxSize

Validates that a FormControl file is smaller than a specific size in bytes.

ExtValidators.fileMinSize

Validates that a FormControl file is larger than a specific size in bytes.

ExtValidators.fileExtension

Validates that a FormControl file has a specific extension.
