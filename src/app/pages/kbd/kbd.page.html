The KBD (keyboard) component can be used to indicate user's keyboard input inside elements such as inputs, tables, text, and other.

Automatic KeyboardEvent translation

KBD provides a feature that automatically converts KeyboardEvent.code into a human-readable format.

This feature is case sensitive, so if you type it out by hand, remember to use PascalCase, just like KeyboardEvent does.

Full key names

By default, KBD will display a shortened name of the key. This can be changed using the full attribute. In some cases, however, there is no difference.

Dynamic keys values

KBD allows to set the displayed key dynamically using the key attribute.

Keyboard shortcuts

Ardium UI exposes the ArdiumKbdShortcut component which streamlines the process of displaying keyboard shortcuts.

KBD Shortcut allows to use all the same customization options as the standard KBD.
